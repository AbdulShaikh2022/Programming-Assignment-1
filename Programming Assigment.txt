4. Write a Python program to swap two variables?
Answer - var1 = 60
var2 = 50

print('Before swap:\nvar1 = {} and var2 = {}'.format(var1, var2))

temp = var1
var1 = var2
var2 = temp
print('\nAfter swap:\nvar1 = {} and var2 = {}'.format(var1, var2))
